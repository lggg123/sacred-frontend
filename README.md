## Getting started

**Requirements:**

- Node.js version 18.0.0 or higher

**Setup:**

1. Install dependencies:
```bash
npm install
```

2. Link to your Shopify store:
```bash
npx shopify hydrogen link
```

3. Pull environment variables:
```bash
npx shopify hydrogen env pull
```

## Building for production

```bash
npm run build
```

## Local development

```bash
npm run dev
```
